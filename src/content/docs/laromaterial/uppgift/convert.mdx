I denna uppgift ska du skriva ett program som omvandlar ett numeriskt värde. Du tränar på användarinmatning, villkorssatser (if/elif/else), datatyper och enkel felhantering.



## Förkunskaper

Du har jobbat allt material i "[kmom01](/kmom/kmom01)".



## Introduktion

Du skall nu göra ett program som räknar ut ett pris efter rabatt och skatt. Användaren ska skriva in priset och programmet ska räkna och skriva ut det slutgiltiga priset. Programmet ska också hantera felaktig input för att programmet inte ska krascha.

Avsluta med att skriva ut det omvandlade värdet med max två decimaler.

Det är viktigt att ditt program följer instruktionerna i kraven annars kommer inte testerna fungera och då rättas inte din inlämning.



## Krav

Lösningar som använder konstruktionerna list/dict comprehension eller lambda funktioner godkänns inte i denna kurs.

Varje uppgift ska lösas med grundläggande Python konstruktioner som vi går igenom i kursen. Egenskapade funktioner får användas först i kmom03 och listor får användas först i kmom04.

1. Skriv din kod i filen `src/kmom01/convert/convert.py`.

```bash
# Ställ dig i roten av kurskatalogen python
touch src/kmom01/convert/convert.py
cd src/kmom01/convert
ls
```

2. `convert.py` skall innehålla kod för att omvandla ett värden, som beskrivs i texten ovanför.

3. Gör ett `input()` anrop för att få värdet som ska omvandlas.
    - Om användaren skriver in ett värde som inte är endast siffror så ska programmet skriva ut ett felmeddelande och avsluta.
        - Felmeddelandet måste innehålla texten `Invalid value`.
        ```python
        Hello and welcome to the price converter!
        Enter value to convert: A
        Invalid value, please enter a number.
        ```
    - Om det är endast siffror, omvandla det till ett heltal, `int`.
4. Om input var ett tal ska du nu räkna ut det nya priset.
        ```python
        Hello and welcome to the price converter!
        Enter value to convert: 100
        The final price of 100.0 after 10 kr discount and 20% tax is: 108.0 kr
        ```

5. Skriva ut original värdet och det omvandlade värdet. Det nya priset ska vara avrundat till max två decimaler. **Obs!** det måste vara ett **mellanslag** direkt efter originalvärdet och det omvandlade värdet i utskriften.

7. Koden ska validera med Ruff.

    ```bash
    # Kommandot funkar oavsett var du står i ditt repo
    uv run ruff check
    ```

1. Testerna som finns ska passera.

    ```bash
    # om du står i katalogen src/kmom01/convert i ditt repo
    uv run task test kmom01/convert
    # om du står i root katalogen i ditt repo
    uv run task test kmom01/convert/
    ```


1. Ladda upp din kod till ditt repo på GitHub. Gör flera små commits när du jobbar med uppgiften!

    ```bash
    git add src/kmom01/convert/convert.py
    git commit -m "Klart med uppgiften convert"
    git push
    ```


## Extrauppgift

- Använd try-except för att hantera felaktig inmatning (t.ex. om användaren skriver in bokstäver i stället för ett tal).
    - Du får inte använda bas felet `Exception`, du specificera vilket typ av fel du vill fånga.
